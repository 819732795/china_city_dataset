# 中国城市地名及区域划分数据集

该数据集可以和中国统计年鉴或者行业数据结合使用。主要包括以下特征：

- City: 城市中文名称缩写
- City_FullName: 城市中文全称
- City_EN: 城市英文名称
- Province: 省中文名
- Province_EN：省英文名
- Region：区域划分（西、南、东、北）
- Tier：城市分级（一线、新一线、二线、三线、四线、五线）
